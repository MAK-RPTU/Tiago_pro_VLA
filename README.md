# Tiago_pro_VLA
This repository is used to test OpenVLA on a tiago robot

- Kyungki Data folder has Presentation
- openvla is a sample example to test VLA
- ros2_ws has nodes to run openvla and moveit for tiago

NOTE: To run openvla we need hugging face token and models.


Please read the NOTES file to setup docker setup of tiago robot and also the presentation slide as a second reference.
